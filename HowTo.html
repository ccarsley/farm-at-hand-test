Add Field

To Field/Activity Management

  • Select Field/Activity Management
  • Select +Field
  • Enter Field, Land Location and Total Acres. Optional information – Crop Type and Rent/Own

Add Activity

Through Field/Activity Management

  • Select field from Field List
  • Select +Activity button
  • Select Date and Activity and Continue

From Navigation Menu

  • Select +Activity from Navigation Menu
  • Select Field, Date and Activity and Continue

From Calendar

  • Select desired date
  • Select +Activity
  • Select Field and Activity, and Continue

Add Alert

Through Calendar

  • Select +Alert
  • Select desired date and input title

Through Navigation Menu

  • Select +Alert
  • Select desired date and input title

Add Bin Yard

to Storage

  • Select Storage
  • Select +Bin
  • Enter Name and Location

Add Bin

to existing bin yard

  • Select an entered Bin Yard
  • Select +Bin, enter Name, Description, and Size
  • Enter Name and Location

Add Grain Bag

to Commodity Storage

  • Select Storage
  • Select +Bag
  • Enter Name, Description, Size, and Field

Add Equipment

to Equipment Management

  • Select Equipment Management
  • Select +Equipment and category
  • To add information, select piece of equipment from list and select the Edit button

Add Maintenance Record

  • Select Equipment Management
  • Select piece of equipment
  • Select Maintenance Records
  • Select +Maint. Record, choose desired Date, Category and Type

Add Part

to Parts List

  • Select Equipment Management
  • Select piece of equipment
  • Select Parts
  • Select +Part, choose desired Date, Category and Type

Add Commodity Contract

  • Select Contracts
  • Select +Contract

Add Commodity Delivery

  • Select Delivery
  • Select +Delivery

Check Commodity Status

  • Select Commodity Status
  • It will display by commodity and variety the amount in Storage, under Contract and the amount left to market

Add Yield

to Yields

  • Select Yields
  • Select +Yield and choose from list of fields
  • Enter Weights and Save

Manage Users

  • Select Settings
  • Select Manage Users

Add new users

  • Select +User
  • Enter required information and set permission levels

Edit users

  • Select Edit from main screen to delete users or select specific user to edit their information or permission levels

Message Center

New messages notifications will appear next to the Message Center

Send a new message

  • Select Message Center
  • Select +Message
  • Enter Subject and Message and Save
  • Sync app to send (if sending from the app)

Reply to message

  • Select Message Center
  • Select message thread from list
  • Select Reply
  • Enter message and Save
  • Sync app to send (if sending from the app)

Changing Units of Measure

  • Select Settings
  • Select Preferences
  • Select either Regions, Units or Change Ind. Units of Measure

Sync information between devices

  • The Sync button is located in all app platforms
  • Once information has been entered on any of the supported devices, use the Sync button to ensure all information is saved and updates across all devices

Delete/Change Information

App

  • Select the Edit button to delete or change already entered information
  • Swipe across information to be deleted, select Delete

Website

  • Select Message Center
  • Use the “x” symbol to delete the corresponding information
  • Use the Edit button to make changes to existing information
